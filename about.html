Who We Are

About Our Group

We are a team of students from the Cloud Computing course, working together on Practical Lab 1 – Virtualization and Static Website Deployment.

The Project

Project Description

This project is part of our Cloud Computing Practical Lab 1. Our goal was to explore the fundamentals of virtualization using Oracle VM VirtualBox and demonstrate static website deployment through GitHub Pages. The website you're viewing right now is our deployment built collaboratively by our group using HTML, CSS, and JavaScript, and hosted for free on GitHub Pages. Through this lab, we gained hands-on experience with virtual machines, version control, team collaboration workflows, and cloud-based hosting.
